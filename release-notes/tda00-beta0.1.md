# TDA00 beta0.1 发布说明

这是 `Muv-Luv Unlimited: THE DAY AFTER TDA00` 的简体中文测试补丁计划说明。

## 版本定位

- 非官方、非商业同人补丁。
- 仅供正版 Steam 玩家测试交流。
- 不包含游戏本体，不提供破解，不修改 exe，不操作存档。
- 当前阶段已冻结 JP baseline，尚未完成完整翻译。

## 本版基准

- 按游戏实际 `egpack` / XML 调用提取 JP 文本。
- 固定 3713 条 JP 文本槽的调用顺序、speaker、ruby、来源 egpack。
- 确认被调用但 JP 为空 0、重复 ID 0、重复 XML 调用 0、英文整句混入 JP 0。
- 确认并同步 368 条 TDA00 专有名词 / 术语候选。

## 已知风险

后续翻译和回写阶段仍需继续检查错译、语病、繁体字 / 日文汉字残留、说话人错位、空字幕、重复字幕或 `Text ID Not Found` 类问题。

请反馈时尽量附上截图、章节、前后台词上下文。
