# 工具总览

这里保存可以随补丁仓库公开的可复用工具。工具按资源格式和用途分组，使用前请先确认目标游戏版本、资源路径和输入 CSV 是否匹配。

## 目录

- `fpd/`：TDA00-03 外层 FPD 资源容器的探测、诊断和筛选提取工具。
- `egpack/`：TDA00-03 内层 EGPACK 多语言表的精确提取、写回和验证工具。
- `rio/`：photonflowers / photonmelodies 使用的 RIO / CRsa 提取、写回、差分工具。
- `font/`：中文字体覆盖和兼容性检查工具。

## 使用原则

- 先提取文本并生成可审计 CSV，再按 `id` / 资源文件 / 场景定位修改。
- 提取时原样保留控制符用于审计；中文写回不加入 `\n`、`\r` 或真实 CR/LF，由引擎自动换行。
- `\p`、`\w`、`\f` 不是手动换行符，是否保留必须按其结束、等待或纯控制用途判断。
- 不做英文兜底、旧中文兜底或模糊匹配兜底。
- 补丁发布优先分发差分或安装包，不直接分发完整游戏资源文件。
- 一次性审计脚本、交接提示词、本地分支脚本、AI 翻译脚本和发布操作笔记只保存在本地内部目录，不放入公开 GitHub 树。
