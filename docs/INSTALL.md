# 安装说明

## 安装前准备

1. 确认已经在 Steam 购买并安装对应章节。
2. 关闭正在运行的游戏。
3. 从 [Releases](https://github.com/imnotsureyi-sys/muvluv-series-steam-cn-patch/releases) 下载对应章节的最新压缩包。
4. 解压压缩包后，优先阅读压缩包内随附的安装说明。

## 推荐安装方式

优先使用发布包内的一键安装脚本。脚本只写入本地用户缓存目录，不修改 exe，不修改 Steam 原始 `pack.bin`，也不操作存档。

如果 Windows 弹出安全提示，请确认文件来自本仓库发布页后再运行。

## 手动安装

不同章节的发布包内容可能略有差异。手动安装时，请以对应发布包内的说明为准。

通常需要做的是：

1. 找到发布包中的补丁文件。
2. 按说明放入对应的本地缓存或补丁目录。
3. 启动游戏并检查标题、设置界面和第一段文本是否显示为中文。

## 安装后检查

安装完成后建议检查：

- 标题或章节名是否显示为中文。
- 设置界面是否显示中文文本。
- 开场字幕或第一段对话是否正常显示。
- 是否出现空字幕、方块字、错位台词或 `Text ID Not Found`。

如果出现问题，请截图并记录章节、场景位置和前后台词上下文。
