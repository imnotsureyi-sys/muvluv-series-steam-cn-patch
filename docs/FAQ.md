# 常见问题

## 这个补丁包含游戏本体吗？

不包含。本项目只提供非官方简体中文补丁，请支持正版游戏。

## 需要破解游戏吗？

不需要。本补丁不提供破解，也不修改 exe。

## 会改 Steam 原始文件吗？

补丁设计上不修改 Steam 原始 `pack.bin`，也不操作存档。安装脚本只写入本地用户缓存目录。

## 为什么是测试版？

TDA00-03 已经可以下载使用，但仍可能存在错译、语病、术语不一致、缺字、字幕图片缺漏或显示槽错位等问题。测试版会根据实机反馈继续修正。

## 发现问题怎么反馈？

请尽量提供：

- 问题截图。
- 对应章节。
- 大致场景位置。
- 前后台词上下文。
- 使用的补丁版本。

反馈群：273626767。

## GitHub 下载慢怎么办？

可以稍后重试，或等待网盘镜像。历史版本和最新版本都以 GitHub Releases 为准。

## 可以转载吗？

请先阅读 [NOTICE.md](../NOTICE.md)。转载时不要打包游戏本体、破解文件或未授权资源，并请保留项目来源和免责声明。
